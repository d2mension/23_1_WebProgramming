  • 데이터 분석가

      데이터 분석가란 데이터를 분석하여 이해한 후 이를 보고서나 시각화 자료 등으로 바꾸어 사용 가능한 인사이트를 제공한다. 데이터 분석가는 조직의 경영 전략 수립, 문제 해결, 미래 트렌드 예측 등을 위한 데이터의 효율적 활용에 핵심적인 역할을 한다.
      데이터 분석가는 많은 양의 데이터 간의 패턴과 관계를 파악하고 분류, 예측, 군집화 등을 해야 하기 때문에 통계학 지식과 머신러닝 지식 등이 필요하다. 또한 비즈니스 의사결정에 직접적으로 영향을 미치므로 자신이 속한 비즈니스 현장에 대한 이해력과 문제 해결, 다른 이들과의 소통 능력 등이 요구된다.
      이전에는 데이터의 정리와 문서 작성을 위해 엑셀을 많이 사용하였으나, 현재는 수집되는 데이터의 양이 방대해지고 종류도 다양해지면서 더 이상 엑셀로 감당할 수 없는 수준이 되었다. 따라서 데이터의 분석과 시각화에 필요한 R, Python 등의 프로그래밍 언어와 함께 예측, 분석의 대상이 되는 서비스 이용에 대한 활용도를 수집하는 도구 등도 알고 있어야 한다.

      데이터 분석가가 하는 일
    • 비즈니스 요구사항 이해
    • 데이터 수집
    • 데이터 정제 및 구조화
    • 데이터 패턴 분석 및 식별

  • 데이터 과학자

      데이터 과학자란 통계분석, 딥러닝, 기계학습 등을 통해 데이터를 수집·관리하고 이를 분석하여 문제에 대한 해결책을 제공하고 개선한다. 데이터 분석가와 비슷한 업무를 진행하나, 데이터 수집과 정제 과정을 거친다는 점에서 조금 더 개발자에 가까운 일을 한다.
      데이터 과학자는 다양한 데이터를 다루고 그 데이터를 목적에 맞게 탐색, 분석해야 하므로 데이터 분석을 위한 통계 등의 수학적 지식, 데이터 전처리와 모델 생성을 위한 코딩 능력 등이 뒷받침되어야 한다. 또한 다양한 부서와의 협업이 필요하므로 소통 능력 또한 필수적으로 요구된다.
      현재 다양한 오픈 소스 소프트웨어와 관련 도구들이 만들어지고 공개되며 매우 빠르게 트렌드가 변화하고 있으나, 일반적으로는 SQL, Python, R 등의 다양한 프로그래밍 언어를 사용한다. 또한 데이터 과학자가 되기 위해서는 일반적으로 통계 모델링, 데이터 관리, 데이터 시각화, 머신 러닝 등을 공부하고 실습한다.

      데이터 과학자가 하는 일
    • 데이터 세트 분석
    • 데이터 정리
    • 데이터 시각화
    • 통계적 추론 생성 및 학습 모델 개발
